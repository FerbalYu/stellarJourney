# Game Design

## 概述

"星際征途" 是一款**基于格子地图的回合制 roguelike 地牢探险游戏**。

- **视角**: 俯视 Tile-based
- **地图**: 程序化 BSP 算法生成多层地牢
- **视野**: 递归阴影投射 + 迷雾系统
- **主题**: 科幻星际探索

## 核心循环

```
进入楼层 → 探索地图 → 收集资源/战斗 → 找到楼梯 → 下降至更深层
    ↑                                                      ↓
    └──────────────── 死亡 / 通关 ←──────────────────────────┘
```

## 当前功能状态

| 功能 | 状态 | 备注 |
|------|------|------|
| 地牢地图生成 | ✅ 已实现 | BSP + 房间 + 走廊 + 楼梯 + 门 |

| 敌人系统 | ❌ 未实现 | 无 AI，无战斗逻辑 |
| 物品系统 | ❌ 未实现 | 无装备，无消耗品，无背包 |
| 存档系统 | ❌ 未实现 | 无持久化 |
| 音效 | ❌ 未实现 | main-menu.js 有 playSound 占位 |

## 地牢设计

### 地图参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| width | 80 | 地图宽度（格） |
| height | 60 | 地图高度（格） |
| minRoomSize | 4 | 最小房间尺寸 |
| maxRoomSize | 12 | 最大房间尺寸 |
| bspDepth | 5 | BSP 递归深度 |
| roomMargin | 2 | 房间边距 |
| totalLevels | 10 | 总层数 |

### 层类型设计

| 层数 | 类型 | 特点 |
|------|------|------|
| 1F | entry | 起始层，较宽敞安全 |
| 3F, 6F, 9F | treasure | 宝箱层，更多奖励 |
| 5F, 10F | boss | 首领层，强敌挑战 |
| 其他 | normal | 标准探险层 |

### 地图元素

- **房间**: 矩形空间，随机尺寸（受 min/maxRoomSize 约束）
- **走廊**: 三种风格 — simple(直线)、L(L型弯折)、S(S型弯曲)
- **门**: 走廊与房间交界处
- **楼梯**: 上行(▲)和下行(▼)，连接层与层
- **墙壁**: 自动围绕地面生成

## 视野系统设计

### 三种视野状态

| 状态 | 含义 | 渲染 |
|------|------|------|
| VISIBLE | 当前在视野半径内 | 完全显示 |
| REMEMBERED | 曾经探索过，但当前看不见 | 半透明显示 |
| HIDDEN | 从未探索 | 完全黑色 |

### 三种视野模式

| 模式 | 效果 |
|------|------|
| NORMAL | 标准视野范围 |
| DARKVISION | 无视光照，更大视野 |
| BLIND | 仅自身一格可见 |

### 与地图的集成方案（待实现）

1. 从 `dungeon.js` 的 TILE.WALL 生成 `Segment` 障碍物
2. 将障碍物注册到 `FogOfWar.addObstacle()`
3. 玩家移动时调用 `FogOfWar.setObserverPosition()`
4. 渲染时用 `FogOfWar.render()` 绘制迷雾遮罩

## UI/UX 设计

### 主菜单 (已实现)

通过 `src/scripts/main-menu.js` 控制：
- 键盘导航（↑↓←→ + Enter）
- 鼠标悬停高亮
- 入场动画
- 设置面板（音量、画质、语言）

### 游戏内 HUD（待实现）

- 小地图（右上角）
- 生命/能量条（左上角）
- 物品快捷栏（底部）
- 消息日志（底部滚动）
- 当前层数显示

## 技术设计决策

| 决策 | 理由 | 文档 |
|------|------|------|
| BSP 生成 | 保证连通性、可控房间尺寸 | [decisions/002-bsp-over-cellular-automata.md](decisions/002-bsp-over-cellular-automata.md) |
| 递归阴影投射 | O(n) 复杂度、支持任意障碍物形状 | `src/visibility/` |
| 双模块系统 | 历史遗留，计划迁移 ESM | [module-migration.md](module-migration.md) |
| 无游戏引擎 | 纯 Canvas 2D，轻量可控 | 项目初期阶段 |
